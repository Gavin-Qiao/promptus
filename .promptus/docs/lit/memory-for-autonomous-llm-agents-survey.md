# Memory for autonomous LLM agents survey

"Memory for Autonomous LLM Agents" (survey, arXiv:2603.07670, Mar 2026). Surveys the agent-memory field and names the open frontier explicitly: memory as a "hypothesis ledger and evidence accumulator" requiring "uncertainty-aware memory" — which it says current systems "handle poorly or not at all." Consolidation value: independent confirmation that Promptus's target (a status-calibrated research ledger the agent reasons over) is the field's acknowledged GAP, not a solved problem — so the effort is well-aimed. Lesson: the moat to build is exactly the uncertainty/status layer; the storage plumbing is commodity.
