# Unlinked references

In Roam Research ("unlinked references") and Obsidian ("unlinked mentions"), the tool surfaces notes that NAME a concept in prose but never link it, so the author can draw the missing edge. A purely lexical, human-judged signal -- no embeddings, no ML. This is the direct ancestor of Promptus's kb-graph suggest: suggest is unlinked-references with IDF weighting plus a shared-source signal, kept suggest-only so the human draws the wikilink. Same discipline as the gate (see [[the-gate]]) -- the script proposes by lexical match; relevance stays the human's call. See [[the-scriptable-graph-layer]].
