# ARA agent-native research artifacts

ARA — "The Last Human-Written Paper: Agent-Native Research Artifacts" (arXiv:2604.24658, Apr 2026, Orchestra Research). A file-based `ara/` package: `logic/claims.yaml` of falsifiable claims each carrying an EPISTEMIC STATUS, a `trace/` exploration graph that preserves dead ends, and an end-of-turn recorder with a Maturity Tracker (knowledge events staged -> crystallized). This is the closest published prior art to Promptus's thesis — file-based, status-tagged, dead-ends-kept — and it postdates the Promptus design (independent convergence). Difference to consolidate against: ARA is a per-paper reproducibility artifact (one package per paper), not a cross-linked markdown knowledge web spanning a research programme, and it is a research prototype, not an installable tool. Lesson: study its claims.yaml schema and the staged->crystallized maturity model as a reference for our finding/status lifecycle; our ledger DEADEND kind already does part of ARA's trace/.
