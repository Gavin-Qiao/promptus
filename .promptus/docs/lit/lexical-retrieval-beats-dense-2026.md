# Lexical retrieval beats dense 2026

Two mid-2026 results that lexical retrieval matches or beats dense for our regime. "BM25 beats SOTA dense on precise-terminology corpora; hybrid+rerank best" (arXiv:2604.01733, Apr 2026). "Agentic grep generally beats vector retrieval for code/agents — tool-presentation dominates" (arXiv:2605.15184, May 2026), mirroring frontier coding agents (Claude Code, Cursor) dropping vector search. Consolidation value: empirical support that for a small, dense, precise-vocabulary corpus with the LLM as ranker, grep + the wikilink graph is current practice, not a compromise. Caveat: the real gap is synonym/paraphrase recall — the cheapest patch when it bites is a reranker, not a vector index.
