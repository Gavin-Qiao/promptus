# EviBound claims-ledger and semantic laundering

EviBound (arXiv:2511.05524, Nov 2025) is the closest IMPLEMENTED artifact to Promptus's moat: an autonomous-research governance framework with a "Claims Ledger" tagging claims VERIFIED / REFUTED / ABSTAINED / INADMISSIBLE_UPDATE to gate false task-completion claims. But it gates claim ADMISSION (assert vs. abstain), not the GRADED assertiveness of admitted prose, and it operates over task-completion claims, not citations to stored knowledge units — so the assertiveness<=status check remains unbuilt. Pair with "Semantic Laundering in AI Agent Architectures" (arXiv:2601.08333, Jan 2026), which names the exact failure mode Promptus guards against: epistemic-status ELEVATION without epistemically-relevant inference (a tool boundary is not epistemic warrant). Together they bound the moat: the idea is in the air (status ledgers + the named failure), but no one enforces graded assertiveness against a cited unit's curated status.
