# The gate

kb-add enforces a HYBRID gate over three separate facets — KIND (the act), STATUS (the claim's epistemic state), and RELATION (a typed link). The curated library (finding/lit/memory) is STRICT: an off-vocab kind/status, a lit unit with no source, or an empty title is refused with the allowed set. The lab-notebook ledger is PERMISSIVE: an off-vocab kind/status is warned about but still written, so recording stays low-friction (friction is what made the old append script drift). No single ontology covers "research events + epistemic statuses", so the gate enforces a small grounded core and warns elsewhere — see [[vocab-grounding-no-single-standard-recommend-a-hybrid-gate]] and [[header-beats-vector]].
