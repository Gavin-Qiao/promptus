# Header beats vector

A hand-written header is a human-authored retrieval key: the model reads it natively, it is transparent, and it carries substrate and status. Better than an embedding for a small, dense, status-tagged corpus; embeddings earn their place only at the papers-scale crossing. See [[markdown-is-the-graph]] and [[the-gate]].
