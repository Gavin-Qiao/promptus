# GraphRAG

Microsoft GraphRAG (Edge et al., 2024): build an entity graph, partition it into hierarchical communities (Leiden), pre-generate a summary per community, then answer global "sensemaking" queries by map-reduce over those summaries. A deferred Promptus roadmap item (report section 5: community summaries) -- not built. A graph-based successor to vector RAG like [[kag-knowledge-augmented-generation]], but aimed at global themes rather than multi-hop precision.
